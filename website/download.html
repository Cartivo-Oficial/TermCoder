Command line

Prefer the terminal? One line.

The CLI installs from npm and runs the same agent on every OS. Install once, then type term in any project folder.

$ npm install -g @termcoder/tui 
# PowerShell or CMD
npm install -g @termcoder/tui
# then, in any folder:
term

Requires Node.js 18+. No account needed — run /setup inside the app to connect a model or pick a local one.

New here? The docs walk you from first run to making it your own.
