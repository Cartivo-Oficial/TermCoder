// INSTALL

One command, every platform.

No accounts, no API keys, no telemetry. The CLI installs from npm and runs the same on Linux, macOS, and Windows. Prefer a window? Download the app — Node is bundled, nothing else to set up.

01
01INSTALL FROM NPM

One command.

Needs Node.js 18 or newer. No Node? Grab the desktop app instead — it bundles its own.

$npm install -g @termcoder/tui
02
02RUN IT

Type term anywhere.

Open a terminal in any project folder and start. Two commands install — term and termcoder — they do the same thing.

~/my-project
$ term
● termcoderfree/auto  free · no key
❯ _
03
03TRUST THE FOLDER

Once per folder.

On first open, TermCoder asks whether you trust the current folder before it reads or runs anything in it. Answer once; it remembers per folder.

trust
Trust the files in this folder?
❯ yes, this is my project
  no, exit
04
04START, OR CONNECT A MODEL

Nothing required.

It runs on a free, keyless model out of the box. Want faster, stronger answers? Connect one — all free options included:

graduate when you want
❯ /upgrade        # free Gemini key, 2 steps
❯ /login-claude   # Claude Pro/Max (experimental)
❯ /login-chatgpt  # ChatGPT Plus/Pro (experimental)
❯ /key google <key>  # or paste one
05
05ASK YOUR FIRST TASK

Plain language.

Describe what you want. It reads the code, proposes edits, and runs commands — asking before anything that changes your machine.

~/my-project
❯ add input validation to the signup form,
  then run the tests
✓ read signup.ts
✓ edit signup.ts
✓ npm test  → passing
