Install

Up and running in a minute.

The CLI installs from npm and works the same on Windows, macOS, and Linux. Prefer a window? Download the desktop app →

1

Install from npm

One command, on any system:

$ npm install -g @termcoder/tui 
# PowerShell or CMD
npm install -g @termcoder/tui

Requires Node.js 18 or newer. Verify with node --version.

2

Run it in your project

Open a terminal in any folder and type:

term

Two commands are installed — term and termcoder — they do the same thing.

3

Trust the folder

On first open, TermCoder asks whether you trust the current folder before it reads or runs anything in it. Answer once and it remembers per folder.

4

Start with no key — or connect a model

Nothing required here: TermCoder runs on a free, keyless model out of the box. Want faster, stronger answers?

❯ /upgrade                        # connect a free Gemini key in two steps
❯ /setup                          # the full guided walkthrough
❯ /key anthropic sk-ant-...        # or set a key yourself

Prefer local? Install Ollama, pull a model, and pick it under "Local" in /model.

5

Ask your first task

Describe what you want in plain language. TermCoder reads the code, proposes edits, and runs commands — asking before anything that changes your machine.

❯ add input validation to the signup form and run the tests

That's it.

Learn the commands, modes, and shortcuts in the docs — or grab the desktop app.
