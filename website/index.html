Why TermCoder

A terminal agent with real tools, and nothing in your way.

An agent loop that reads, edits, and runs — wrapped in a keyboard-driven interface that stays out of the way.

Built for the terminal

Slash commands, @file mentions, a live model picker, multi-line input, and a trust prompt on first run.

Any model, or local

Anthropic, OpenAI, or Gemini with your key — or point it at a local Ollama model. No account, no lock-in.

Build and Plan modes

Plan mode reads and proposes without touching files; Build mode carries the change out. Switch with shift+tab.

Sub-agents and skills

Delegate focused work to specialist sub-agents. Reusable skill playbooks load on demand, only when needed.

Kind to your context

Old tool output is elided and long results are capped, so a long session stops re-billing everything it has read.

A study mode inside

Switch to TermExplorer and the same tool becomes a patient tutor for schoolwork. No programming required.

TermExplorer

A study companion in the same tool.

Pick the TermExplorer model and TermCoder explains step by step, summarizes your notes, and builds flashcards, practice quizzes, and study plans — in your language.
