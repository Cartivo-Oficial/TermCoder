---
name: study-plan
description: Build a realistic revision schedule for an exam or deadline
---
Create a study plan the student can actually follow.

1. Ask for (or use what's given): the subject and topics, the exam/deadline date, and how
   many hours a day they can realistically study. Don't stall if some are missing —
   assume sensible defaults and say what you assumed.
2. Break the material into topics and spread them across the days, using **spaced
   repetition** (revisit hard topics more than once) and leaving the last day for review
   and practice questions.
3. Output a simple day-by-day table: date → topics → activity (learn / practice / review)
   → rough time.
4. Add 3–4 concrete study tips for the subject, and offer to make flashcards or a practice
   quiz for any topic.
