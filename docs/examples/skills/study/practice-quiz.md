name: practice-quiz
description: Generate a practice quiz on a topic, with an answer key
---
Create a practice quiz to test the student's understanding.

1. Use the topic (or notes) the student names. Aim for 5–10 questions that cover the key
   ideas at a mix of difficulties.
2. Vary the format: some multiple-choice, some short-answer, and at least one "explain
   why" question that checks real understanding, not just recall.
3. Present the **questions first** (numbered), then an **answer key** below with a one-line
   explanation for each — so they can test themselves before checking.
4. Offer to mark their answers if they reply with them, and to focus the next quiz on
   whatever they got wrong.
